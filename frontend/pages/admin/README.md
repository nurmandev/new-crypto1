# Admin Dashboard

This folder contains the admin panel components and pages for managing the platform.

## Structure

```
admin/
├── components/        # Reusable admin components
│   ├── AdminHeader.tsx
│   ├── AdminSidebar.tsx
│   ├── StatsCard.tsx
│   └── index.ts
├── AdminDashboard.tsx # Main admin dashboard page
└── README.md
```

## Components

### AdminHeader

Main header component for admin pages with notifications, settings, and logout.

### AdminSidebar

Navigation sidebar with menu items for different admin sections.

### StatsCard

Reusable card component for displaying statistics with trends.

## Pages

### AdminDashboard

Main admin dashboard page with overview statistics and recent activity.
